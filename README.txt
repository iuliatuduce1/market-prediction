# Market Predictor Application

This repository contains the Jupyter notebook for the Market Predictor Application. Also, you can find an example for the data, stocks.csv .

## Table of Contents

1. Description
2. Installation
3. Usage
4. Contributing

## Description

This project serves as a predictor for stock market trends using decision tree regression and linear regression models.
 The application uses historical data of various stocks to predict the closing price for the next 100 days.

## Installation

To set up the project locally, follow these steps:

* Clone the repository:

        git clone https://github.com/iuliatuduce1/market-predictor.git

* Navigate to the project directory:

        cd market-predictor

## Usage

The project consists of a Jupyter notebook `MarketAnalysis.ipynb` that contains all the code for data preprocessing,
model training, and prediction generation. The `stocks.csv` file is the dataset used in this project.
